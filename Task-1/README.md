# Task 1 – Data Cleaning and Preprocessing

## Objective
Clean and preprocess a raw dataset by handling missing values, removing duplicates, standardizing text, formatting dates, and verifying data types using Python (Pandas).

## Tools Used

- Python
- Pandas
- Google Colab
- Excel

## Steps Performed

- Imported dataset
- Checked missing values
- Filled missing values
- Removed duplicate rows
- Renamed column headers
- Standardized categorical values
- Converted date columns
- Verified data types
- Exported cleaned dataset
